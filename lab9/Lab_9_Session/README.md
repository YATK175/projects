# Lab_9_Session — Laboratory work 9, Variant 3 Books

This branch continues the Fastify Books API and adds Redis integration for caching and distributed rate limiting.

## Commands

```bash
npm install
npm run db:generate
npm run db:migrate
npm run seed
npm run dev
```

Run Redis and MySQL locally before starting the server. For Lab_9_Session and Lab_9_JWT use auth endpoints before POST/PATCH/DELETE item operations.
